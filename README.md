[IN DEVELOPEMENT]

A chess game state manager that is intended to be variant friendly.

Notes:
Uses webpack-cli

Build:
npm run build

Run:
npm start

Build Then Run:
npm run super


Standard Chess Grid:

    A   B   C   D   E   F   G   H
  ---------------------------------
0 | r | n | b | q | k | b | n | r |
  ---------------------------------
1 | p | p | p | p | p | p | p | p |
  ---------------------------------
2 |   |   |   |   |   |   |   |   |
  ---------------------------------
3 |   |   |   |   |   |   |   |   |
  ---------------------------------
4 |   |   |   |   |   |   |   |   |
  ---------------------------------
5 |   |   |   |   |   |   |   |   |
  ---------------------------------
6 | P | P | P | P | P | P | P | P |
  ---------------------------------
7 | R | N | B | Q | K | B | N | R |
  ---------------------------------

NOTE: white's pieces are upper case and black's pieces are lower case.
