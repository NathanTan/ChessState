# ChessState

### A chess game state manager that is intended to be variant friendly (eventually).


Example of usage: https://github.com/NathanTan/ChessStateExample

Notes:
Uses webpack-cli

Build:
npm run build

Run:
npm start

Build Then Run:
npm run super


Standard Chess Grid:
```
               Black
  ---------------------------------
0 | r | n | b | q | k | b | n | r |
  ---------------------------------
1 | p | p | p | p | p | p | p | p |
  ---------------------------------
2 |   |   |   |   |   |   |   |   |
  ---------------------------------
3 |   |   |   |   |   |   |   |   |
  ---------------------------------
4 |   |   |   |   |   |   |   |   |
  ---------------------------------
5 |   |   |   |   |   |   |   |   |
  ---------------------------------
6 | P | P | P | P | P | P | P | P |
  ---------------------------------
7 | R | N | B | Q | K | B | N | R |
  ---------------------------------
    A   B   C   D   E   F   G   H
               White
```
